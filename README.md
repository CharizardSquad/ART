# FireSpin API
Welcome to FireSpin API

## Highlights 
FireSpin API is a web application that functions as a developer tool to reference the overall health of various API's for current and future product development. 

## Built With
- React
- TypeScript
- React Router
- Chart.js
- Cypress
- Jest
- Sequelize
- PostgreSQL
- Node
- Express
